# Requirements: Shadowdark Character Sheet

**Defined:** 2026-03-21
**Core Value:** A player can open their character, see all their stats with full bonus breakdowns, and manage their inventory slot-by-slot — everything needed at the table that Shadowdarklings doesn't provide.

## v1.1 Requirements

### File Management

- [ ] **FILE-01**: User can save the current character to a .sdchar file via the File menu
- [ ] **FILE-02**: User can load a character from a .sdchar file via the File menu
- [ ] **FILE-03**: User can import a Shadowdarklings.net JSON export via the File menu

### Talents & Spells

- [ ] **TLNT-01**: User can view and edit a Talents/Spells free-text area in the Notes tab, above the Notes editor

## Future Requirements

### Polish

- **PLSH-01**: Gear slot utilization shown as visual progress bar
- **PLSH-02**: Character sheet auto-saves on every change

### Ledger

- **LEDG-01**: User can view full GP/SP/CP transaction history
- **LEDG-02**: User can add ledger entries with description and amount

### Bonus Management

- **BNUS-01**: User can toggle individual bonus sources on/off per stat
- **BNUS-02**: Disabled bonuses are excluded from stat totals

## Out of Scope

| Feature | Reason |
|---------|--------|
| Character builder (auto talent tables, ancestry bonuses) | Manual entry only — out of scope by design |
| Online sync or cloud saves | File-based only |
| Multi-character roster UI | OS file management handles character switching |
| Shadowdarklings-compatible JSON export | Import only |
| Spell management beyond free-text | Free-text Talents/Spells area is sufficient |
| Dice roller | Not a character sheet concern |
| Automatic attack derivation from weapon stats | Attacks are free-form text |
