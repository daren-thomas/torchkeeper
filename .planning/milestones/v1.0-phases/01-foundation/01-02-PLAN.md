      - character.BaseSTR > 0 (at least one base stat is non-zero)

    Test 2 — Import_UsesRolledStats:
      Inline JSON with both "stats":{"STR":18} and "rolledStats":{"STR":14,...} populated.
      Assert character.BaseSTR == 14 (the rolledStats value, not 18).

    Test 3 — Import_Currency_TopLevelFields:
      Inline JSON with "gold":10,"silver":5,"copper":3 and a "ledger":[{"goldChange":999}].
      Assert character.GP == 10, character.SP == 5, character.CP == 3.
      (Top-level fields win over ledger.)

    Test 4 — Import_Currency_LedgerFallback:
      Inline JSON with no gold/silver/copper top-level fields (omit them entirely), ledger with
      two entries: goldChange sums to 15, silverChange to 3, copperChange to 0.
      Assert character.GP == 15, character.SP == 3, character.CP == 0.

    Test 5 — Import_AcBonuses:
      Inline JSON with bonuses array: one entry with bonusTo="AC:+2", one with bonusTo="DEX:+1".
      Assert character.Bonuses has 2 entries.
      Assert exactly one BonusSource has BonusTo starting with "AC:".
      Assert exactly one BonusSource has BonusTo starting with "DEX:".
      (Both go to the Bonuses list — AC prefix is stored as-is, not filtered out.)

    Test 6 — Import_UnknownFields_Ignored:
      Inline JSON containing extra fields not in ShadowdarklingsJson ("futureField":"value","unknownArray":[1,2]).
      Call ImportAsync. Assert no exception is thrown. Assert result is not null.

    Test 7 — Import_CurrentHP_EqualsMaxHP:
      Inline JSON with "maxHitPoints":12.
      Assert character.MaxHP == 12 and character.CurrentHP == 12.

    GREEN phase — implement ShadowdarklingsImportService to make all 7 tests pass.

  <implementation>
    Replace the empty ShadowdarklingsImportService stub with a real implementation.

    Class structure:
    - Private static readonly JsonSerializerOptions ImportOptions with PropertyNameCaseInsensitive=true,
      AllowTrailingCommas=true, ReadCommentHandling=JsonCommentHandling.Skip.
      (Unknown fields are silently ignored by System.Text.Json default — no extra option needed.)
    - Public async Task<Character?> ImportAsync(Stream jsonStream) method.

    ImportAsync logic:
    1. Deserialize the stream to ShadowdarklingsJson using ImportOptions.
       Wrap in try/catch JsonException — return null on parse failure.
    2. If result is null, return null.
    3. Currency: GP = sdJson.Gold ?? (sdJson.Ledger?.Sum(e => e.GoldChange) ?? 0).
       Same pattern for SP (Silver/SilverChange) and CP (Copper/CopperChange).
    4. Map all BonusSource entries from sdJson.Bonuses — include ALL of them regardless of BonusTo prefix.
       BonusTo values starting with "AC:" represent AC contributors; the Bonuses list holds both stat and AC
       contributors (differentiated by prefix at display time in Phase 2). Do not filter by prefix here.
       Map: Label = sdBonus.SourceName, BonusTo = sdBonus.BonusTo, SourceType = sdBonus.SourceType,
       GainedAtLevel = sdBonus.GainedAtLevel.
    5. Map SdGearItem list to GearItem list: Name, Slots, ItemType = sdItem.Type, Note.
    6. Map SdMagicItem list to MagicItem list: Name, Slots, Note.
    7. Return new Character with all fields populated. CurrentHP = MaxHP = sdJson.MaxHitPoints.
       Use RolledStats for base stat values (RolledStats?.STR ?? 0, etc.).
       Attacks = sdJson.Attacks ?? [].

    Do NOT create a new JsonSerializerOptions per call (performance pitfall — use the static readonly instance).
    Do NOT read sdJson.Stats (that property does not exist on the DTO and would double-count bonuses).

    After implementation, update MauiProgram.cs to uncomment the ShadowdarklingsImportService registration:
    ```csharp
    builder.Services.AddSingleton<ShadowdarklingsImportService>();
    ```
