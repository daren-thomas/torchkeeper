    }

**NotesPage.xaml:**
```xaml
<?xml version="1.0" encoding="utf-8" ?>
<ContentPage xmlns="http://schemas.microsoft.com/dotnet/2021/maui"
             xmlns:x="http://schemas.microsoft.com/winfx/2009/xaml"
             x:Class="TorchKeeper.Views.NotesPage"
             Title="Notes">
    <Grid Padding="16">
        <Editor Text="{Binding Notes, Mode=TwoWay}"
                Placeholder="Your notes here..."
                VerticalOptions="Fill"
                HorizontalOptions="Fill"
                AutoSize="TextChanges" />
    </Grid>
</ContentPage>
```

Notes:
- `Grid` as root (not VerticalStackLayout) so the Editor fills the full page height via `VerticalOptions="Fill"`
- `AutoSize="TextChanges"` allows the Editor to grow with content while staying within the page
- No ScrollView wrapper needed — Editor handles its own scrolling when it fills the parent
- No formatting controls (plain text only, per Claude's Discretion)
