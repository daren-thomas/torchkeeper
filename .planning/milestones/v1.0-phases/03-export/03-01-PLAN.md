From CharacterViewModel (computed properties the DTO must mirror):
- TotalSTR = BaseSTR + sum of bonuses where BonusTo starts with "STR:"
- ModSTR = floor((TotalSTR - 10) / 2)
- GearSlotTotal = max(TotalSTR, 10)
- CoinSlots = max(GP-100,0)/100 + max(SP-100,0)/100 + max(CP-100,0)/100 (integer division)
- GearSlotsUsed = sum of gear item slots + CoinSlots
